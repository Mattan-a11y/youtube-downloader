# YouTube Downloader

A simple desktop application for downloading YouTube videos, built with Python.

---

## Features

- Download YouTube videos directly to your desktop
- Real-time progress bar during download
- Download status and speed display
- Threaded downloads — the UI stays responsive while downloading

---

## Requirements

- Python 3.8 or higher
- The following Python libraries:

```
customtkinter
yt-dlp
```

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Mattan-a11y/youtube-downloader.git
cd youtube-downloader
```

2. Install dependencies:

```bash
pip install customtkinter yt-dlp
```

3. Run the application:

```bash
python "Youtube downloader.py"
```

---

## Usage

1. Launch the application
2. Paste a YouTube URL into the input field
3. Click **Download**
4. The file will be saved to your Desktop

---
